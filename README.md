## monorepo 使用示例

### npm

切换到 npm 分支：

```
git checkout npm
```

安装依赖：

```
npm install
```

运行：

```
node ./packages/app
```

### Yarn Berry

切换到 yarn berry 分支：

```
git checkout yarn-berry
```

安装依赖：

```
yarn install
```

运行：

```
yarn node ./packages/app
```
